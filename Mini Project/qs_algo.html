Quick Sort Algorithm

Algorithm

        function quickSort(arr, low, high) {
            if (low < high) {
                let pi = partition(arr, low, high);

                quickSort(arr, low, pi - 1);
                quickSort(arr, pi + 1, high);
            }
        }

        function partition(arr, low, high) {
            let pivot = arr[high];
            let i = low - 1;

            for (let j = low; j < high; j++) {
                if (arr[j] <= pivot) {
                    i++;
                    let temp = arr[i];
                    arr[i] = arr[j];
                    arr[j] = temp;
                }
            }

            let temp = arr[i + 1];
            arr[i + 1] = arr[high];
            arr[high] = temp;

            return i + 1;
        }

Information

Quick Sort is a highly efficient, in-place sorting algorithm that divides an array into smaller sub-arrays, then recursively sorts those sub-arrays. It is one of the most commonly used sorting algorithms in computer science.

Explanation

The Quick Sort algorithm works by selecting a pivot element from the array and partitioning the other elements into two sub-arrays, according to whether they are less than or greater than the pivot. The sub-arrays are then sorted recursively, and the pivot element is placed in its correct position. This process continues until the entire array is sorted.
