Merge Sort Algorithm

Algorithm

        function mergeSort(arr) {
            if (arr.length <= 1) {
                return arr;
            }

            const mid = Math.floor(arr.length / 2);
            const left = arr.slice(0, mid);
            const right = arr.slice(mid);

            return merge(mergeSort(left), mergeSort(right));
        }

        function merge(left, right) {
            let result = [];
            let leftIndex = 0;
            let rightIndex = 0;

            while (leftIndex < left.length && rightIndex < right.length) {
                if (left[leftIndex] < right[rightIndex]) {
                    result.push(left[leftIndex]);
                    leftIndex++;
                } else {
                    result.push(right[rightIndex]);
                    rightIndex++;
                }
            }

            return result.concat(left.slice(leftIndex), right.slice(rightIndex));
        }

Information

Merge Sort is a divide-and-conquer sorting algorithm that works by dividing the unsorted list into smaller sublists, sorting those sublists, and then merging them to produce a sorted list. It is a stable sorting algorithm known for its reliable performance.

Explanation

The Merge Sort algorithm divides the input array into two halves and recursively sorts each half. The sorted halves are then merged to produce a single sorted array. The merging process involves comparing elements from the two sublists and selecting the smallest element to be placed in the final result. This process continues until the entire array is sorted.
