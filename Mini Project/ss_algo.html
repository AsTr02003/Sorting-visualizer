Selection Sort Algorithm

Algorithm

        function selectionSort(arr) {
            let n = arr.length;

            for (let i = 0; i < n - 1; i++) {
                let minIndex = i;

                for (let j = i + 1; j < n; j++) {
                    if (arr[j] < arr[minIndex]) {
                        minIndex = j;
                    }
                }

                if (minIndex !== i) {
                    let temp = arr[i];
                    arr[i] = arr[minIndex];
                    arr[minIndex] = temp;
                }
            }
        }

Information

Selection Sort is a simple and inefficient sorting algorithm that divides the input array into two sub-arrays: the sorted part on the left and the unsorted part on the right. It repeatedly selects the minimum element from the unsorted part and places it at the beginning of the sorted part.

Explanation

The Selection Sort algorithm works by iterating through the input array, selecting the minimum element from the unsorted portion, and swapping it with the element at the beginning of the unsorted portion. This process is repeated until the entire array is sorted. Although simple, it is not suitable for sorting large lists due to its time complexity.
