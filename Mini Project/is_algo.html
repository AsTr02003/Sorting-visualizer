Insertion Sort Algorithm

Algorithm

        function insertionSort(arr) {
            for (let i = 1; i < arr.length; i++) {
                let key = arr[i];
                let j = i - 1;
                while (j >= 0 && arr[j] > key) {
                    arr[j + 1] = arr[j];
                    j = j - 1;
                }
                arr[j + 1] = key;
            }
        }

Information

Insertion Sort is a simple sorting algorithm that builds the final sorted array one item at a time. It is much less efficient on large lists than more advanced algorithms such as quicksort, heapsort, or merge sort.

Explanation

The Insertion Sort algorithm works by dividing the input array into two parts: the left part, which is initially sorted, and the right part, which is unsorted. The algorithm iterates through the unsorted part, taking one element at a time and placing it in its correct position within the sorted part. It repeats this process until the entire array is sorted.
